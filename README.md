## Get Started

Follow the steps below to properly run this application.

#### Clone the repository:

```bash
git clone https://github.com/gasscoelho/quizersoon-flutter.git
```

```bash
cd quizersoon-flutter
```

#### Start the application:

```bash
flutter run
```
